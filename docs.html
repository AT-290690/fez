Fez

Simple, typed, lisp programming language

Here you will learn everything you need to know about Fez.

  • It has types and they are mostly infered (no need to define them).
  • Everything is an expression (everything can be assigned to a variable).
  • Tree shaking of functions from standard library (imports automatically and only the functions needed).
  • Tail call and Memoized optimizations on recursions using a simple prefix.
  • It has only Floats, Arrays and Functions.
  • Booleans are the numbers 0 and 1.
  • Strings, Objects, Sets, Trees - these are implemented using only the Array.
  • There is an interpreter written in JavaScript.
  • There is a compiler that compiles to JavaScript.
  • You can always try it on the Online Playground .

This language is so simple that an interpreter for it can be implemented in any other language

For example there are python, rust and even fez interpreters!

The one here can easily run them on the web where you can share your programs with others via a link.
